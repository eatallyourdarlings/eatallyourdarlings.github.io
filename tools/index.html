tools

these are self-contained browser-based tools (single HTML files), which means you can use them here or download the page and still use them with your browser offline. please feel free to edit them to suit your needs.

flashbang warning: these tools don't have dark mode support, unlike the rest of this site.

  • dither - shrink images using cute patterns. forked from eli oat
  • image data - turn an image into a string of useful nonsense. handy for single-page HTML zines
  • pomo - a minimal timer for getting things done. forked from monoxane (I hardly use this anymore, I just put on a nice album as a container of time).

also check out ../etc/ for some of my experiments, and small and nearly silent dot com for more cool tools!
