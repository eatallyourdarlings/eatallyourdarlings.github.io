I'm keeping a copy of this beautiful little open-source (RMIT licensed) tool on this site for easy access because I use it a lot, and because I intend to study its code and fork it to add some fancy stuff one day.

notes

  • one dot in the grid is 15px in CSS.
  • when exporting a vector, it's the size of the canvas that determines the SVG width/height. this means that if you zoom in, it'll export just the zoomed part of the vector.
  • if you start drawing from the the top left, you can resize as you go without much issue.
  • using the arrow keys to move around and keyboard shortcuts below is generally a nice quick way to make a glyph, but I don't know if you can move points without using a mouse to click-drag.

README

Dotgrid is a grid-based vector drawing software designed to create logos, icons and type. It supports layers, the full SVG specs and additional effects such as mirroring and radial drawing. Dotgrid exports to both PNG and SVG files.The application was initially created for internal use, and later made available as a free and open source software.

mouse click-drag

  • Hold Shift to drag a segment.
  • Hold Alt to drag all segments.
  • Right-click to delete a vertex.
