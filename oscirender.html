an open-source oscilloscope software that is compatible with Blender and VST and MIDI and scriptable with Lua!

OsciBlender

  • open Blender (I use 3.6) and Oscirender.
  • in Blender render properties tab, connect to Oscirender.
  • Set render aspect to square (e.g. 512px).
  • Oscirender needs line art, so add a grease pencil based on object/collection/scene.
  • go to line art’s modifier tab, bake.
  • each time u change modifier settings again u want to bake again.

works well with nice lil lowpoly models :]
