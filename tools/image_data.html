turn an image into a string of useful nonsense




upload a file, copy the data and paste it in an image element's src attribute. right click + inspect element on the image below for an example.

this is useful for embedding small images inside a HTML page, for example if you're making one-file web-zines. not recommended for larger images i.e. a megabyte or more. if it looks too big on this page it's probably too big for a li'l zine!
