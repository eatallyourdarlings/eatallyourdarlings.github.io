pandoc wiki spell on 'nix, where www is a symlink from notes to the folder for a website:


for f in *.md; do 
d=$(date -r $f '+%Y-%m-%d'); 
pandoc -f markdown-auto_identifiers -t html --wrap=none --lua-filter=www/pandoc/filter.lua --template=www/pandoc/template.html -o www/"${f%.*}".html -M name="${f%.*}" -M mdate=$d $f; done
